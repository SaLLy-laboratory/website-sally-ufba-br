Introduction to R Programming: A Beginner's Guide to Data Science and Statistical Computing. To be determined.

    SaLLy's introductory course on R programming is a beginner-level course that teaches the basics of R programming language, which is widely used in data science and statistical computing. The course covers essential concepts such as data types, variables, functions, loops, and conditional statements. It also provides an introduction to data manipulation and visualization using R packages such as ggplot2 and dplyr. The goal of the course is to provide students with a solid foundation in R programming, enabling them to write their own R scripts, work with data, and produce basic visualizations.
